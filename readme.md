识别58同城发布人手机号
===================================
###用途
将![58同城手机号示例](http://image.58.com/showphone.aspx?t=v55&v=EE3486214C1A7AE787A29CF26A69C207C "58同城手机号示例")转换为17092646159

### 目录说明
- template 保存模版
- test 保存临时图片，若程序跑不出来，可将该目录清空后再次运行
- test_backup 保存做实验用的数据
- main.py 主程序

### 环境
- python 2.7

### 依赖的库
- cv
- numpy
- PIL

### 安装教程    
- 安装setuptools，在https://pypi.python.org/pypi/setuptools#files下载setuptools-17.0.zip，，解压后在命令行中运行python setup.py install即可。（装pip前必装）
- 安装pip：在https://pypi.python.org/pypi/pip#downloads下载pip-7.0.3.tar.gz (md5, pgp)，，解压后在命令行中运行python setup.py install即可。添加环境变量：C:\Python27\ArcGIS10.2\Scripts
- 下载opencv-3.0.0.exe安装，直接双击，什么都不用做，会在你当前exe文件所在目录里生成一个opencv文件夹。将C:\Program Files (x86)\opencv\build\python\2.7\x86下的cv2.pyd复制到C:\Python27\ArcGIS10.2\Lib\site-packages中。
- 下载numpy，在http://sourceforge.net/projects/numpy/files/NumPy/，或者http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy，pip install "numpy-1.9.2+unoptimized-cp27-none-win32.whl"
- 在python中import cv2即可。
- 下载opencv_python-2.4.11-cp27-none-win32.whl，pip install opencv_python-2.4.11-cp27-none-win32.whl后，即可import cv
